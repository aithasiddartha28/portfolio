BloodCoin — Smart Blood Donation & Reward Platform

Full-Stack AI Web App

VisionAI — Real-Time Object Detection

AI · TensorFlow.js · No API Key · 100% Free

A browser-based real-time object detection app that uses your device camera to identify and label objects live using the EfficientDet machine learning model — completely free with no API key required. Runs entirely in the browser.

🔴 Problem

Object detection apps usually require paid APIs or complex backend setups. Making AI computer vision accessible directly in the browser with zero cost was the challenge.

🟢 Solution

Integrated TensorFlow.js with EfficientDet Lite models to run real-time object detection entirely in the browser — no server, no API key, no cost.

Outcome: Detects 90+ object types live from your camera with bounding boxes, confidence scores, detection log, FPS counter, and 3 model variants (Fast / Balanced / Accurate).
